System Requirements

  • Linux (Ubuntu or Debian) preferred or Windows®
  • Minimum 1 TB Storage
  • Minimum 8–16 GB RAM
  • Minimum 8-Core CPU with Multithreading
  • Minimum 8 GB 4-Core GPU

Summary

The Standard System is software that runs a proprietary DB and coding language to unify how records are stored, saved, and used. The future Standard AI will have access to these records, giving it a holistic overview of your digital life.

Standard offers a standard coding language (.stds, .std) to serve custom applications from your Standard instance or to integrate existing solutions with your Standard Device or system. Standard code can run queries and access your records.

Standard comes with out-of-the-box functionality; it supports files, searches, conversational AI, and more. You can remotely access your Standard instance if it is connected to the internet and using a relay.

If your Standard System instance is local (not connected to the internet), you can access it and perform the same online functionality. Standard comes with a corpus which means you can search the internet, even when you're offline.

Installing on Windows

To install the Standard System on Windows, download the installer by clicking here.

  1. Launch the installer
  2. Provide license key
  3. Choose the installation type
    • Cloud – Selecting this installation type means you're subscribing to the public relay access monthly subscription.
    • Custom – This installation type assumes you have a second key to set up a relay version of the Standard System on another device.
    • Relay – Choosing this means you're running the Standard System as a relay on a Windows device.

If Cloud

  1. Copy and paste info

    The Device Serial, User ID, and Chit Token are provided after converting a license to a device via the public relay subscription.

  2. Installer will validate info

If Custom/Local

  1. Set endpoint

    This is the address of the device on the internet that will be running the Standard System in relay mode. The public relay is relay.standardcomputers.net.

  2. Maintain data

    If you provided a User ID, the installer will download and autofill your information to create the user for the Standard System. If not, you'll be prompted to provide the required information.

If Relay

  1. Enter local device info

    Enter the Device Serial, User ID, and Chit Token created by the Custom/Local installation of the Standard System.

  2. Keep calm, carry on

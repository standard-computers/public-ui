Using Standard

Preface

Standard unifies all data at the foundation, so features aren’t separated into apps. Instead, everything is accessed through UI windows called interfaces in the Public UI or through commands. When you log into your device at standardcomputers.net, you see a single screen showing the objects in focus. Start typing to take action—view email, open documents, ask questions, and more. Everything flows through one point because the underlying data and mechanisms share the same source.

Traditional systems rely on app launchers, isolated app settings, separate installers, and countless configuration layers. Standard replaces this fragmentation by defining one underlying model capable of representing any object. The sections that follow explain how each core feature of Standard works and how they interact with the records within your Standard System.

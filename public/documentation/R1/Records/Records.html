Records

Records are rows in a table that are created using the values provided for constraints. Like other databases, there are many ways you can interact with records or perform queries. The succeeding sections will cover each action in depth and how to make the most of your standardized data.

Overview

Records/tables can be accessed by starting a query with a Standard's name or reference in brackets, followed by the command. 

    
    [vehicles] + ("VIN173", "Dodge", "Durango", "Red")

Counting Records

Count records in a Standard table by prepending a record query with count. 

    
    count [vehicles]

Purging Records

To empty a Standard table (remove all records), prepend the table declaration with purge. 

    
    purge [vehicles]
